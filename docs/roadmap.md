Expandability

The structure is designed to be easily expandable: For more comparison pages, create new files like:

app/vs/rev-ai/page.tsx
app/vs/assemblyai/page.tsx
app/vs/descript/page.tsx

For more use case pages, create new files like:

app/use-cases/podcasts/page.tsx
app/use-cases/legal/page.tsx
app/use-cases/healthcare/page.tsx
app/use-cases/education/page.tsx
